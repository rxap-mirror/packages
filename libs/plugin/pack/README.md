# Installation

Add the plugin to your workspace:


```
yarn add @rxap/plugin-pack
```


Configure @rxap/plugin-pack for a project:

```
ng g @rxap/plugin-pack:config [project]
```

*Adds the @plugin-pack:build to the specified project*

# Schematics
